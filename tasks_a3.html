Игра "Быки и коровы"

A. Дана строка длиной больше 2 символов. Нужно напечатать новую строку, равную старой, из которой исключили символ с заданным номером. Цикл использовать нельзя!

Пример:

Вход Выход

'abcde'

2

 'abde'

B. Дана строка из неповторяющихся символов. Нужно случайным образом выбрать из неё заданное число неповторяющихся символов и напечатать в виде строки.

Используйте решение задачи A в качестве функции.

Используйте генератор псевдослучайных чисел из стандартного модуля random. Библиотека подключается следующей строкой в начале программы:

import random

После этого вы можете использовать в своей программе функции из этой библиотеки, например:

n = random.randint(0, 5)   # Функция randint() даёт в ответе случайное целое число из заданного интервала, включая граничные значения.

C. Возможно, сейчас эта игра называется по-другому. Раньше она была известна под названием "Быки и коровы".

Программа загадывает строку из четырёх неповторяющихся цифр (или из заданного набора символов) и предлагает игроку угадать. Если в строке игрока есть загаданные символы, которые стоят на своих местах, программа говорит: "столько-то быков". Если в строке игрока есть загаданные символы, которые стоят не на своих местах, программа говорит: "столько-то коров" (символы, стоящие на своих местах, при подсчёте коров надо исключить!). На каждом шаге игрок предлагает свой ответ, а программа отвечает: "Столько-то быков, столько-то коров." Игра заканчивается, когда строка угадана.

Используйте решение задачи B в качестве функции.
