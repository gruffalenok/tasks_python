Арифметические операции с двоичными числами

Перевод, сравнение, сложение и вычитание

Напишите тестирующую программу по двоичной арифметике для учащихся 7-8 классов, изучающих основы современных цифровых технологий.

Задание:

A. Напишите функцию bin_to_dec(), которая переводит числа из двоичной системы в десятичную. Функция получает на вход строку из символов "0" или "1" - число в двомчной системе счисления, даёт в ответе целое число в десятичной системе счисления.

B.  Напишите функцию dec_to_bin(), которая переводит числа из десятичной системы в двоичную. Функция получает на вход целое число в десятичной системе счисления и число разрядов, выводит строку из символов "0" или "1" заданной длины ("лишние" разряды можно отбрасывать).

С.  Напишите функцию dec_to_bin1(), которая переводит числа из десятичной системы в двоичную. Функция получает на вход целое число в десятичной системе счисления, выводит строку из символов "0" или "1", начинающуюся с "1".

D*. Напишите тестирующую программу по двоичной арифметике, которая будет предлагать пользователю переводить числа из двоичной системы счисления в десятичную, из десятичной в двоичную, сравнивать числа в двоичной и десятичной системах и проверять ответ. Используйте генератор псевдослучайных чисел.

E.  Напишите функцию compare(), которая сравнивает два числа в двоичной системе. Функция получает на вход две строки не обязательно равной длины из символов "0" или "1", выводит символ ">", "<" или "=", в зависимости от результата сравнения. Преобразования типов данных использовать нельзя. (То есть нельзя сравнивать числа, только символы.)

F. Напишите функцию sum_bin(), которая складывает два двоичных числа. Функция получает на вход две строки одинаковой длины из символов "0" или "1", выводит строку - сумму этих чисел в двоичной системе. Арифметическими операциями пользоваться нельзя!!! Логический оператор if и операторы сравнения использовать нельзя! Можно использовать логические операторы and, or, not, преобразование типов, цикл, сложение строк.

    Указание:
  1. Чтобы логические функции работали правильно, нужно каждый символ введённой строки перевести сначала в int, а затем в bool.
  2. Напишите функцию, складывающую два бита. Она должна получать на вход два логических значения и возвращать люгическое значение - младший бит суммы. Используя её, несложно сложить три бита. Если вы умеете складывать три бита, значит, вы можете сложить числа любой длины.

б) (более простой вариант - если сложный совсем не получается) Можно использовать таблицу сложения в двоичной системе и логический оператор.

G. Напишите функцию, которая прибавляет к двоичному числу 1. (Сложение с числом "000...01" не является решением.) Функция получает на вход строку из символов "0" или "1" - число в двомчной системе счисления, даёт в ответе строку из символов "0" или "1" - число в двоичной системе счисления, увеличенное на 1. Арифметическими операциями пользоваться нельзя!!! Логический оператор if и операторы сравнения использовать нельзя! Можно использовать логические операторы and, or, not, преобразование типов, цикл, сложение строк.

H. Напишите функцию subtract_bin(), которая находит разность двух двоичных чисел. Функция получает на вход две строки одинаковой длины из символов "0" или "1", выводит строку - разность этих чисел в двоичной системе. Арифметическими операциями пользоваться нельзя!!! Логический оператор if и операторы сравнения использовать нельзя! Можно использовать логические операторы and, or, not, преобразование типов, цикл, сложение строк.

Указание: Если перевести вычитаемое в дополнительный код, то можно использовать сложение.

I*. Напишите тестирующую программу по двоичной арифметике, которая будет предлагать пользователю примеры на сложение и вычитание двоичных чисел и проверять результат. Примеры должны записываться "столбиком".
