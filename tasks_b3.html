Тестирующая программа по арифметике

Напишите тестирующую программу по арифметике для младшеклассников, которая должна вывести пример, считать с клавиатуры ответ, проверить его правильность и вывести итог: правильно решено, или нет.

A. Напишите пять программ, проверяющих решение примеров разного вида:

  • a. на сложение,
  • b. на вычитание (результат должен быть >= 0),
  • c. на умножение,
  • d. на деление (результат должен быть целый),
  • e. на деление с остатком (результатом являются два числа - частное и остаток, остаток должен быть меньше, чем делимое и >=0).

Условная инструкция

Чтобы примеры каждый раз получались разные, используйте генератор псевдослучайных чисел из стандартной библиотеки (модуля) random.

Пример:

import random
a = random.randint(1, 10)
b = random.randint(1, 10)
print(a, '+', b, '=', end = ' ') # вместо перехода на следующую строку будет пробел
c = int(input())
# дальше должна быть проверка

Подсказка: случайный пример на вычитание легко получить из случайного примера на сложение, а случайный пример на деление так же легко получается из случайного примера на умножение (и тогда всё точно разделится).

Оператор while

Оператор while, так же как и оператор if, содержит условие - логическое выражение, которое может принимать истинное или ложное значение, и "тело" - список инструкций, которые выполняются в том случае, если условие истинно. Разница в том, что "тело" условного оператора выполняется один раз, и после этого программа переходит к выполнению следующих инструкций. Если условие оператора while истинно, то после выполнения "тела" оператора программа возвращается к условию и снова его проверяет. Соответственно, инструкции из "тела" оператора while повторяются до тех пор, пока условие не станет ложным. (А программисту нужно позаботиться о том, чтобы это когда-нибудь произошло.)

Прочтите статьи про цикл while. Ещё статья на informatics.msk.ru.

B. Скопируйте и выполните следующую программу:

import random
score = 0
right = False
while not right:
    a = random.randint(1,10)
    b = random.randint(1, 10)
    print(a, '+', b, '=', end=' ')
    c = int(input())
    if a + b == c:
        right = True
    else:
        print('Неправильно. Попробуй решить ещё один пример:')
print('Верно!')

Что делает эта программа?

1) Измените программу так, чтобы программа предлагала решать примеры, пока на них дают правильные ответы. А в случае ошибки прекращала свою работу.

2) Измените программу так, чтобы она прекращала работу в случае ошибки и писала число набранных очков - количество правильно решённых примеров. 

3) Измените программу так, чтобы она прекращала работу в случае ошибки, или если правильно решено три примера.

Функции

Если мы хотим создать тестирующую программу, которая предлагает ученику решать много примеров, то будет удобно выделить в отдельный блок инструкции, предъявляющие и проверяющие один пример. Потом можно будет вызвать и выполнить этот блок в любом месте программы. 

Скопируйте и выполните следующий пример:

import random    # В начале программы импортируем все необходимые модули

# Теперь определяем все функции. Этот код сразу выполняться не будет.

def addition():  # Определение функции: специальное слово def, название функции со скобками и двоеточие.
    # Дальше с отступом идёт "тело" функции.
    # Это инструкции, которые выполнятся, если функция будет вызвана из основной программы.
    a = random.randint(10, 99)
    b = random.randint(10, 99)
    print(a, '+', b, '=', end=' ')  # вместо перехода на следующую строку после = будет пробел
    answer = int(input())
    if answer == a + b:
        print('Верно')
    else:
        print('Не верно')

# Начало основной программы
print('Решите пример')
addition()     # Вызов функции из программы. Теперь начнут выполняться инструкции, составляющие "тело" функции.
addition()     # Вызываем функцию ещё раз
print('Вы закончили тестирование')

Возврат значений из функции. Оператор return

Недостаток  программы из предыдущего задания в том, что основная программа не "знает", правильно или нет решены примеры. Проблему можно решить, используя оператор return, который может возвращать из функции в программу некоторое значение.

Скопируйте в новый файл и выполните следующий пример:

import random    # В начале программы импортируем все необходимые модули

# Теперь определяем все функции. Этот код сразу выполняться не будет.

def addition():  # Определение функции: специальное слово def, название функции со скобками и двоеточие.
    # Дальше с отступом идёт "тело" функции.
    # Это инструкции, которые выполнятся, если функция будет вызвана из основной программы.
    a = random.randint(10, 99)
    b = random.randint(10, 99)
    print(a, '+', b, '=', end=' ')  # вместо перехода на следующую строку после знака = будет пробел
    answer = int(input())
    if answer == a + b:
        print('Верно')
		return True
    else:
        print('Не верно')
        return False
	print('Эта строчка никогда не будет напечатана')

# Начало основной программы
print('Решите пример')
ans1 = addition()     # Вызов функции из программы. Переменная ans1 равна True или False, в зависимости от возвращённого функцией значения
ans2 = addition()     # Вызываем функцию ещё раз
ans3 = addition()     # Вызываем функцию ещё раз
print('Вы закончили тестирование. Результат:', ans1, ans2, ans3)

F. Измените эту программу так, чтобы программа печатала в конце, сколько примеров решено правильно, а сколько - неправильно.

Подсказка: логические значения можно складывать. При этом True интерпретируется как 1, а False - как 0.
