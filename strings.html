Строки и срезы

В режиме командной строки задайте значение переменной abc = "ABCDE". Меняя значение индекса, заполните (на листочке) следующую таблицу. Перечеркните выражения, приводящие к ошибке.

  • Что вычисляет функция len(abc) ?
  • Начиная с какого индекса нумеруются символы в строке?

A. Напишите программу, которая считывает в переменную string строку длиной больше 3 символов, вычисляет и печатает результат:

  1. длину строки,
  2. двумя способами предпоследний символ строки, Подсказка: можно считать символы от начала строки, а можно - с конца.
  3. двумя способами первый символ строки,
  4. символ, который стоит в середине строки или справа от середины (для нечётного и чётного количества символов),
  5. символ, который стоит в середине строки или слева от середины.

В режиме командной строки задайте значение переменной abc = "ABCDE". Меняя значение индекса, заполните (на листочке) следующую таблицу. Перечеркните выражения, приводящие к ошибке.

B. Напишите программу, которая считывает в переменную string строку длиной больше 3 символов и

  1. Печатает эту строку без первого и последнего символа.
  2. Делит строку пополам и печатает отдельно первую и вторую половинки (если число символов нечётно, то вторая часть может быть на один символ короче).
  3. Считывает с клавиатуры число n < len(string) и печатает строку без символа с номером n.
  4. Печатает символы в обратном порядке ( подсказка )
  5. Определите, является ли строка палиндромом,

C. Напишите программу, которая считывает с клавиатуры две строки с неповторяющимися символами и печатает:

  1. сколько символов из второй строки содержится в первой строке,
  2. все символы из второй строки, которые входят в первую строку,
  3. все символы из второй строки, которые НЕ входят в первую строку.
  4. Напишите программу, которая определяет, есть ли в данной строке повторяющиеся символы (используйте срезы). Подсказка: используйте цикл for и оператор in.

D. Напишите программу, которая считывает с клавиатуры строку, состоящую из нескольких слов, разделённых пробелами (между каждой парой слов ровно один пробел). Программа должна напечатать каждое слово на новой строке.

Указание: используйте цикл for

Е. Напишите программу, которая считывает с клавиатуры строку с неповторяющимися символами и число k меньше длины строки. Программа должна вывести случайные неповторяющиеся k символов из этой строки.

Подсказка: Чтобы вычислить случайный символ, используйте функцию randint(). Чтобы символы не повторялись, удалите выбранный символ из исходной строки (задача В пункт 3) и повторите операцию нужное число раз.

F. Напишите программу, которая считывает с клавиатуры строку с неповторяющимися символами,

  1. переставляет в ней символы случайным образом и печатает результат,
  2. определяет, сколько символов новой строки оказались на тех же местах, что и в исходной строке.
